# 🧀 CheeseGames

> **Train your brain while having fun.**

CheeseGames es una colección de minijuegos desarrollados con **HTML5, CSS3 y JavaScript** diseñados para entrenar la memoria, la lógica, la velocidad mental y el aprendizaje mediante experiencias rápidas, ligeras y accesibles directamente desde el navegador.

---

## 🌟 Características

- 🎮 Minijuegos rápidos y entretenidos
- 🧠 Entrenamiento de memoria y lógica
- 🌍 Juegos educativos de geografía
- ⚡ Sin instalaciones ni dependencias
- 📱 Diseño responsive para escritorio, tablet y móvil
- 🌐 Funciona directamente en cualquier navegador moderno
- 🚀 Código ligero y de alto rendimiento

---

## 🎮 Juegos

| Estado | Juego | Descripción |
|--------|--------|-------------|
| 🚧 | 🌍 Flag Memory | Memoriza banderas de países y mejora tu memoria visual. |
| ⏳ | 🧩 Find the Differences | Encuentra todas las diferencias antes de que termine el tiempo. |
| ⏳ | 🧠 Memory Challenge | Ejercita tu memoria con patrones e imágenes. |
| ⏳ | ⚡ Reflex Click | Pon a prueba tu velocidad de reacción. |
| ⏳ | 🌎 World Quiz | Trivia de países, capitales y cultura general. |
| ⏳ | 🔤 Word Puzzle | Juegos de palabras y vocabulario. |
| ⏳ | 💻 Tech Matrix | Rompecabezas inspirados en programación y tecnología. |
| ⏳ | 📦 Inventory Match | Organiza objetos utilizando lógica y memoria. |

---

## 🚀 Tecnologías

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Canvas API
- CSS Animations

## 🎯 Objetivo

Crear una plataforma de minijuegos educativos, modernos y ligeros que ayuden a desarrollar habilidades cognitivas mientras el usuario se divierte.

Cada juego estará diseñado para mejorar aspectos como:

* 🧠 Memoria
* 🎯 Concentración
* ⚡ Velocidad mental
* 🌍 Conocimiento general
* 🔍 Observación
* 🧩 Resolución de problemas
