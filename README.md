# 🧠 BrainLab

BrainLab es una colección de minijuegos desarrollados con HTML, CSS y JavaScript para entrenar la memoria, la lógica y el aprendizaje de forma divertida.

## 🎮 Juegos

- 🌍 Flag Memory (En desarrollo)
- 💻 Tech Matrix (Próximamente)
- ⚙️ Pipeline Panic (Próximamente)
- 📦 Inventory Match (Próximamente)

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎯 Objetivo

Construir una plataforma de juegos educativos, ligeros y accesibles que ayuden a aprender mientras se juega.
